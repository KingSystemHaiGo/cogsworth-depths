# Cogsworth Depths

[中文 README](./README.md)

A **zero-asset, fully procedural** steampunk 2.5D roguelite for the browser.
Not a single image or audio file ships with this project — models, textures, VFX, music and SFX are all generated by code at runtime.

## Play

The build is a single self-contained HTML file (`docs/index.html`) — download and double-click to play offline, or play online via GitHub Pages.

## Controls

| Key | Action |
| --- | --- |
| WASD / Arrows | Move |
| Mouse | Aim (pointer lock) |
| LMB | Fire |
| Space | Dash |
| Shift | Roll (i-frames) |
| Esc | Pause |
| F1 | Dev console (style filters / feel / volume) |

## Game loop

Clear each room of clockwork guards → pick 1 of 3 stackable upgrades → detour to treasure rooms & cog shops → boss (with an enrage phase at 50% HP) → next floor. Floor layouts are seed-generated; enter a seed on the title screen to replay a layout.

Language (中文/EN), music volume and SFX volume can be changed on the settings screen, and are persisted in localStorage.

## Tech stack (procedural-first)

- **Three.js**: 3D scenes with a 2.5D top-down camera; gears/pipes/boilers are all modeled in code (`ExtrudeGeometry` / `LatheGeometry` / `TubeGeometry`)
- **PixiJS v8**: HUD, damage numbers, particles, upgrade UI, screen-space crosshair — bridged onto the 3D world via world-to-screen projection
- **Custom comic render pipeline**: toon shading + back-face silhouette outlines + Fresnel rim light + Sobel ink lines + Bayer halftone dithering + pixelation + split-tone grading
- **Web Audio API**: fully synthesized SFX (shots / explosions / metal clanks / steam) and a tracker-style lookahead-scheduled music engine
- **Balance model**: all numbers live in `src/game/balance.ts` with a polynomial difficulty curve
